Técnicas de reducción dimensional

PCA — Análisis de Componentes Principales

Descomposición espectral de la covarianza. Transforma lineal ortogonal. Componentes ordenados por varianza explicada.

C = (1/n)XᵀX → C·vₖ = λₖ·vₖ → PC_k = X·vₖ
LinealMuy rápidoDeterministaNo captura no-linealidad

t-SNE

Preserva distancias locales entre vecinos. Excelente para visualización. No proyectable a nuevos datos.

min KL(P‖Q) q(ij) ∝ (1+‖yᵢ−yⱼ‖²)⁻¹
✓ Visualización 2D✗ No usar como preprocessing✗ Lento O(n²)

UMAP

Preserva estructura local Y global. Más rápido que t-SNE y proyectable a nuevos datos.

Grafo k-NN ponderado → embedding de baja dimensión
✓ Rápido✓ Proyectable✓ Local + global
from sklearn.decomposition import PCA from sklearn.manifold import TSNE # pip install umap-learn import umap # PCA — cuántos componentes para 95% de varianza pca = PCA(n_components=0.95, random_state=42) X_pca = pca.fit_transform(X_scaled) print(f"Dimensiones: {X_scaled.shape[1]} → {X_pca.shape[1]}") print(f"Varianza explicada: {pca.explained_variance_ratio_.sum()*100:.1f}%")

Algoritmos de clustering

K-Means

min Σᵢ Σₓ∈Cᵢ ‖x − μᵢ‖² E-step: asignar al centroide más cercano M-step: μᵢ = media(Cᵢ)
✓ Rápido O(nkd)✓ Escalable✗ K debe definirse✗ Asume clusters esféricos

DBSCAN

Core point: |N_ε(p)| ≥ MinPts Cluster = conjunto de core points densamente conectados Ruido = puntos no alcanzables (etiqueta -1)
✓ K automático✓ Detecta outliers✗ ε y MinPts sensibles

GMM — Gaussian Mixture Models

p(x) = Σₖ πₖ · N(x | μₖ, Σₖ) EM: E-step (probabilidades) + M-step (parámetros)
✓ Probabilístico✓ Clusters elípticos✗ K debe definirse

Clustering Jerárquico (Agglomerative)

Linkage: single=min(d), complete=max(d) average=mean(d), ward=min(varianza)
✓ Dendrograma✓ Sin K fijo✗ Lento O(n²)

Preprocesamiento para no supervisado

Más crítico que en supervisado: no hay etiquetas que compensen errores en los datos.

StandardScalerFeatures con diferente escala distorsionan distanciasK-Means, DBSCAN, PCA, t-SNE, UMAP
RobustScalerResistente a outliers que inflan la escalaMismos + con outliers
ImputaciónLa mayoría de algoritmos no toleran NaNTodos
OHE / GowerDistancias mixtas para datos categóricosK-Means, DBSCAN

Impacto de outliers en K-Means

Los outliers desplazan los centroides y crean clusters espurios. Tratar antes o usar DBSCAN.

Impacto de outliers en PCA

Inflan la varianza y distorsionan los componentes. Usar Robust PCA o log-transform previo.

Selección del K óptimo

Elbow Method (Inercia)

Graficando WCSS vs K, buscar el "codo" donde la disminución se estabiliza.

WCSS = Σᵢ Σₓ∈Cᵢ ‖x − μᵢ‖²

Silhouette Score

Para cada punto: s = (b−a)/max(a,b). Rango [−1, 1]. Elegir K que maximiza la media.

from sklearn.metrics import silhouette_score s = silhouette_score(X, labels)

BIC / AIC para GMM

Criterios de información que penalizan la complejidad. Elegir K que minimiza BIC.

gmm = GaussianMixture(n_components=k) gmm.fit(X) bic = gmm.bic(X)

K-distancia para DBSCAN

Ordenar distancias al k-ésimo vecino. El codo de la curva indica el eps óptimo.
