¿Qué es Keras?

Keras es la API oficial de alto nivel integrada en TensorFlow desde la versión 2.0. En 2023 lanzaron Keras 3.0 con soporte multi-backend (JAX, PyTorch, TensorFlow). Permite construir y entrenar modelos con muy pocas líneas de código sin sacrificar flexibilidad.

Las tres APIs en detalle

🔵 Sequential API — arquitecturas lineales
import keras from keras import layers model = keras.Sequential([ keras.Input(shape=(784,)), # define explícitamente el input layers.Dense(128, activation='relu', kernel_initializer='he_normal'), layers.BatchNormalization(), layers.Dropout(0.3), layers.Dense(64, activation='relu'), layers.Dense(10, activation='softmax') ], name='mi_clasificador') model.summary() # tabla con capas, shapes y parámetros
model.add() también funcionaDefinir Input() es buena práctica
🟣 Functional API — arquitecturas complejas
inputs = keras.Input(shape=(784,), name='imagen') x = layers.Dense(128, activation='relu')(inputs) x = layers.BatchNormalization()(x) x = layers.Dropout(0.3)(x) x = layers.Dense(64, activation='relu')(x) outputs = layers.Dense(10, activation='softmax')(x) model = keras.Model(inputs=inputs, outputs=outputs, name='functional_classifier')

Permite múltiples inputs (inputs=[img, meta]), múltiples outputs, conexiones residuales y ramas paralelas.

# Ejemplo: conexión residual (ResNet style) def residual_block(x, units): shortcut = x x = layers.Dense(units, activation='relu')(x) x = layers.BatchNormalization()(x) x = layers.Dense(units)(x) x = layers.Add()([x, shortcut]) # skip connection return layers.Activation('relu')(x)
🟢 Model Subclassing — control total
class MiModelo(keras.Model): def __init__(self, num_classes): super().__init__() self.dense1 = layers.Dense(128, activation='relu', kernel_initializer='he_normal') self.bn = layers.BatchNormalization() self.drop = layers.Dropout(0.3) self.dense2 = layers.Dense(num_classes, activation='softmax') def call(self, inputs, training=False): x = self.dense1(inputs) x = self.bn(x, training=training) # comportamiento distinto en train/test x = self.drop(x, training=training) return self.dense2(x) model = MiModelo(num_classes=10) model(tf.zeros((1, 784))) # construir el modelo (necesario antes de summary)
💡 ¿Cuál API usar? Empieza con Sequential para proyectos simples. Usa Functional para producción y arquitecturas con skip connections. Usa Subclassing solo cuando necesites lógica condicional en el forward pass.

Callbacks — controlar el entrenamiento

Los callbacks se ejecutan en puntos específicos del ciclo. Son la forma de monitorear, controlar y personalizar el proceso sin reescribir el loop.

EarlyStopping

keras.callbacks.EarlyStopping( monitor = 'val_loss', patience = 10, restore_best_weights= True, mode = 'min', min_delta = 1e-4 )

Detiene si la métrica no mejora en patience épocas. Restaura los mejores pesos automáticamente.

ReduceLROnPlateau

keras.callbacks.ReduceLROnPlateau( monitor = 'val_loss', factor = 0.5, # LR × 0.5 patience= 5, min_lr = 1e-7, verbose = 1 )

Reduce el LR cuando la pérdida se estanca. Permite empezar con LR alto y refinarlo.

ModelCheckpoint

keras.callbacks.ModelCheckpoint( filepath = 'mejor_modelo.keras', monitor = 'val_auc', save_best_only = True, mode = 'max', verbose = 1 )

Guarda el modelo cuando la métrica mejora. Formato .keras es más robusto que HDF5.

TensorBoard

keras.callbacks.TensorBoard( log_dir = './logs', histogram_freq = 1, # histogramas de pesos write_graph = True, update_freq = 'epoch' ) # Lanzar: tensorboard --logdir ./logs

Visualiza curvas, histogramas y grafos del modelo en tiempo real en el navegador.

LearningRateScheduler

def cosine_schedule(epoch, lr): total = 100 return lr_max * 0.5 * ( 1 + np.cos(np.pi * epoch / total)) keras.callbacks.LearningRateScheduler( cosine_schedule, verbose=0)

Aplica warmup, cosine decay, step decay, etc. según una función personalizada.

Custom Callback

class MiCallback(keras.callbacks.Callback): def on_epoch_end(self, epoch, logs=None): if logs.get('val_auc', 0) > 0.99: print('\n✅ AUC > 0.99 — deteniendo') self.model.stop_training = True def on_train_begin(self, logs=None): print('Iniciando entrenamiento…')

Hooks: on_train/test/predict_begin/end, on_epoch/batch_begin/end.

💡 Combinación recomendada para producción: EarlyStopping (patience=15) + ReduceLROnPlateau (patience=7) + ModelCheckpoint (save_best_only=True)

Componentes personalizados

Custom Layer — heredar de keras.Layer
class MiCapa(keras.layers.Layer): def __init__(self, units, activation='relu', **kwargs): super().__init__(**kwargs) self.units = units self.activation = keras.activations.get(activation) def build(self, input_shape): # Los pesos se crean aquí (se conoce el shape del input) self.w = self.add_weight( shape=(input_shape[-1], self.units), initializer='glorot_uniform', trainable=True, name='w') self.b = self.add_weight( shape=(self.units,), initializer='zeros', trainable=True, name='b') def call(self, inputs, training=False): return self.activation(inputs @ self.w + self.b) def get_config(self): # necesario para serialización cfg = super().get_config() cfg.update({'units': self.units, 'activation': keras.activations.serialize(self.activation)}) return cfg # Uso: capa = MiCapa(64, activation='relu') output = capa(tf.zeros((32, 128)))
Custom Loss — Focal Loss para clases desbalanceadas
class FocalLoss(keras.losses.Loss): """Focal Loss: da más peso a ejemplos difíciles. Ideal para detección de objetos y desbalance extremo.""" def __init__(self, gamma=2.0, alpha=0.25, **kwargs): super().__init__(**kwargs) self.gamma = gamma self.alpha = alpha def call(self, y_true, y_pred): y_pred = tf.clip_by_value(y_pred, 1e-7, 1 - 1e-7) bce = -(y_true * tf.math.log(y_pred) + (1 - y_true) * tf.math.log(1 - y_pred)) p_t = y_true*y_pred + (1-y_true)*(1-y_pred) alpha = y_true*self.alpha + (1-y_true)*(1-self.alpha) return tf.reduce_mean(alpha * (1-p_t)**self.gamma * bce) model.compile(optimizer='adam', loss=FocalLoss(gamma=2.0))
Custom Metric — F1 Score stateful
class F1Score(keras.metrics.Metric): def __init__(self, threshold=0.5, **kwargs): super().__init__(**kwargs) self.threshold = threshold self.tp = self.add_weight('tp', initializer='zeros') self.fp = self.add_weight('fp', initializer='zeros') self.fn = self.add_weight('fn', initializer='zeros') def update_state(self, y_true, y_pred, sample_weight=None): preds = tf.cast(y_pred >= self.threshold, tf.float32) y = tf.cast(y_true, tf.float32) self.tp.assign_add(tf.reduce_sum(y * preds)) self.fp.assign_add(tf.reduce_sum((1-y) * preds)) self.fn.assign_add(tf.reduce_sum(y * (1-preds))) def result(self): p = self.tp / (self.tp + self.fp + keras.backend.epsilon()) r = self.tp / (self.tp + self.fn + keras.backend.epsilon()) return 2 * p * r / (p + r + keras.backend.epsilon()) def reset_state(self): self.tp.assign(0); self.fp.assign(0); self.fn.assign(0) model.compile(optimizer='adam', loss='bce', metrics=[F1Score(name='f1')])

Transfer Learning y Fine-tuning

Reutiliza pesos preentrenados en ImageNet, BERT, etc. Permite obtener excelentes resultados con pocos datos y menos tiempo de entrenamiento.

1

Cargar modelo base preentrenado sin la capa de clasificación original

2

Congelar todas las capas del base model (trainable=False)

3

Añadir nuevo clasificador sobre el base model con tus clases

4

Entrenar solo el clasificador (feature extraction) — LR normal

5

Descongelar las últimas N capas del base model

6

Re-entrenar con LR muy bajo (fine-tuning) — típico: 1e-5

Feature extraction — congelar base model

# Cargar EfficientNetB0 preentrenado en ImageNet base = keras.applications.EfficientNetB0( weights = 'imagenet', include_top = False, # sin el clasificador original input_shape = (224, 224, 3) ) base.trainable = False # ❄️ congelar todos los pesos # Construir nuevo clasificador inputs = keras.Input(shape=(224, 224, 3)) x = base(inputs, training=False) x = layers.GlobalAveragePooling2D()(x) x = layers.Dropout(0.3)(x) outputs = layers.Dense(num_clases, activation='softmax')(x) model = keras.Model(inputs, outputs) model.compile( optimizer = keras.optimizers.Adam(1e-3), loss = 'sparse_categorical_crossentropy', metrics = ['accuracy'] ) model.fit(train_ds, epochs=10, validation_data=val_ds)

Fine-tuning — descongelar capas superiores

base.trainable = True # descongelar todo # Congelar solo las primeras N capas (las más genéricas) fine_tune_at = 100 for layer in base.layers[:fine_tune_at]: layer.trainable = False print(f"Capas entrenables: {len([l for l in base.layers if l.trainable])}") # Recompilar con LR mucho más bajo para no destruir los pesos preentrenados model.compile( optimizer = keras.optimizers.Adam(1e-5), # 100× menor que antes loss = 'sparse_categorical_crossentropy', metrics = ['accuracy'] ) model.fit(train_ds, epochs=10, validation_data=val_ds, callbacks=[cb_early, cb_reduce])
